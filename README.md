# Grocery Management System
## Project Overview

This project is a Python-based inventory management tool designed to categorize and retrieve retail product data. It demonstrates the use of **Python Dictionaries** for efficient data storage and mapping.

## Features
- **Categorized Data:** Separate structures for Vegetables, Fruits, Groceries, and Sports/Toys.
- **Pricing Logic:** Includes currency-specific (₹) pricing and unit measurements (kg, dozen, litre).
- **Scalable Architecture:** Built to allow easy expansion of product categories.
- 
## Technical Implementation
- **Language:** Python 3.x
- **Data Structures:** Utilizes Dictionaries (Hash Maps) for $O(1)$ average time complexity for product lookups.

## Usage
Run the script to display the current inventory:
python inventory_manager.py
